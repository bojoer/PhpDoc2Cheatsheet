Retrieving data from view

From Element view:

$comments = $this->requestAction('/comments/latest');
foreach ($comments as $comment) {
    echo $comment['Comment']['title'];
}

In Controller:

$this->requestAction(
    array('controller' => 'articles', 'action' => 'featured'),
    array('return')
);
